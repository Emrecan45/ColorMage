# ColorMage - Version Exécutable Windows

Bienvenue sur la version **Windows** de ColorMage !  
Cette branche contient la version compilée du jeu au format `.exe`, prête à être lancée sur un système Windows.

---

## 📦 Contenu de la branche

- `dist/game.exe` : L’exécutable principal du jeu.  
- `img/` : Dossier contenant toutes les images nécessaires au jeu.  
- `build/` : Dossier temporaire généré par PyInstaller.  
- `game.spec` : Fichier de configuration PyInstaller.  
- `README.md` : Ce fichier d’informations.
